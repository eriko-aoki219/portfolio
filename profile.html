About

“伝えたい想い”が、
誰かの心に届くように。

1991年生まれ。東京都出身。

これまで事務職として長く働いてきましたが、
「誰かの心に届く表現がしたい」という思いから、
Webデザイナーの道を目指すようになりました。

Webの世界に惹かれたのは、その自由さと、正解のない中で
"本当に伝えたいこと"を形にできる力にあります。

時間・場所・性別・言語といったあらゆる枠を超えて、
まっすぐに届けられる。その可能性に強く魅かれました。

これまでの仕事で身につけた
「要点を整理し、わかりやすく伝える力」は、
今では構成や言葉選びにも活かされています。

今後は、日常の中でふと目にするバナーや広告など、身近なところから、
そっと心に残るようなデザインを届けていきたいです。

モノやコトだけでなく、その背景にある
ストーリーを伝えられるデザイナーを目指しています。

Skill

Web Design
画像の補正、切り抜き、バナー、ロゴの制作が出来ます。
Tools：Photoshop / Illustrator / XD
Web Development
HTML/CSSを使用し、
規則性のあるコーディングを行うことが出来ます。
Tools：Visual Studio Code

Vision

情報をただ「伝える」のではなく、
見た人の中に、静かに残る"気づき"や"余韻"を
届けられるデザインを目指しています。

バナーや広告のような、
日常の中で何気なく目にするものこそ、
ほんの少しの工夫や温度感で
「心に残る」表現に変えられると思っています。

最終的には、
誰かの感情や行動のきっかけとなるような、
社会的なメッセージにも取り組める
デザイナーになることが目標です。
